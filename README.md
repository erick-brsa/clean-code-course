# 🧪 Proyecto de Práctica — SOLID y Clean Code

Este proyecto utiliza **TypeScript** sin frameworks (Vanilla) junto con **Vite**, una herramienta moderna que funciona como *bundler* (empaquetador de módulos) y servidor de desarrollo rápido.
El objetivo es proporcionar una base sencilla para realizar ejercicios enfocados en **Principios SOLID** y **Clean Code**.

---

## 📦 Instalación

Clona el repositorio o descarga el proyecto. Después instala las dependencias:

```bash
yarn install
```

o

```bash
npm install
```

---

## ▶️ Ejecución del proyecto

Para iniciar el servidor de desarrollo ejecuta:

```bash
yarn dev
```

o

```bash
npm run dev
```

Esto iniciará el servidor local de **Vite** con **Hot Module Replacement (HMR)**, lo que permite ver los cambios en tiempo real sin recargar manualmente.

---

## 🎯 Objetivo del proyecto

Este repositorio está pensado para practicar:

* Principios **SOLID** de diseño orientado a objetos.
* Buenas prácticas de **Clean Code**.
* Organización y legibilidad del código.
* Uso básico de **TypeScript** en proyectos modernos.

---

## 🧱 Tecnologías utilizadas

* **TypeScript** — Superset de JavaScript que añade tipado estático.
* **Vite** — Herramienta moderna de desarrollo frontend rápida.
* **Node.js** — Entorno de ejecución para gestionar dependencias.

---

## 📁 Uso del repositorio

Puedes utilizar este proyecto para:

* Implementar ejemplos de cada principio **SOLID**.
* Refactorizar código que viole buenas prácticas.
* Experimentar con patrones de diseño y arquitectura de software.
